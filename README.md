**Latvia Macroeconomic Dashboard (1999–2024)**
**Overview**
Interactive Excel dashboard analyzing macroeconomic indicators of Latvia across:
  1. Real Sector
  2. Monetary Sector
  3. Inflation
  4. Labour Market
  5.External Sector
  6. Fiscal Sector
Built entirely in Microsoft Excel using structured tables, slicers, pivot tables, and dynamic formulas.

**Tools Used**
  1. Microsoft Excel
  2. Structured Tables
  3.Pivot Tables
  4.Slicers
  5.INDEX & MATCH
  6.Custom KPI Cards
  7.Multi-sheet dashboard architecture

**Features**
  1. Year-based slicer control
  2. Fully interactive charts
  3. Clean dark professional theme
  4. Dynamic KPI calculations
  5. Derived indicators (Trade Balance, Fiscal Composition, etc.)

**Key Insights**
  - Trade balance fluctuations across crisis periods
  - Inflation volatility and core divergence
  - Fiscal deficit expansion during downturns
  - Long-term public debt trends
  - Gender wage growth comparison
